# Temperature Conversion Program

This repository contains two C programs that convert temperatures between Fahrenheit and Celsius. These programs demonstrate basic input-output operations and the application of mathematical formulas in C.

## Programs Included

### 1. Fahrenheit to Celsius Converter

This program converts a given temperature in Fahrenheit to its equivalent in Celsius using the formula:


![Fahrenheit to Celsius Converter Demo](images/expanding-cards.png)

#### How it Works:
- The program prompts the user to input a temperature in **Fahrenheit**.
- It then applies the formula to convert the temperature to **Celsius**.
- Finally, it displays the temperature in **Celsius**.

### 2. Celsius to Fahrenheit Converter

This program converts a given temperature in Celsius to its equivalent in Fahrenheit using the formula:

![Fahrenheit to Celsius Converter Demo](images/expanding-cards.png)


#### How it Works:
- The program prompts the user to input a temperature in **Celsius**.
- It then applies the formula to convert the temperature to **Fahrenheit**.
- Finally, it displays the temperature in **Fahrenheit**.

## How to Run the Programs

### Requirements:
- A C compiler (e.g., GCC, Clang) installed on your system.

### Instructions:

1. **Clone or Download the Repository**:
   - You can clone this repository using Git, or simply download the program files to your computer.

2. **Compile the Code**:
   Open a terminal (or command prompt) and navigate to the directory where the C files are located.

   To compile the programs, use the following commands:
   
   For **Fahrenheit to Celsius**:
   ```bash
   gcc fahrenheit_to_celsius.c -o fahrenheit_to_celsius
